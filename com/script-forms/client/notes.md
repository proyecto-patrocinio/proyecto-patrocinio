# Activadores

**Activador 1**
    * función: ``onFormOpen``
    * Ejecución durante el despliegue: ``Principal``
    * Fuente del evento: ``De un formulario``
    * Tipo de evento: ``Al abrirse``
    * Notificar errores: ``Notificarme cada día``

**Activador 2**
    * función: ``onFormSubmit``
    * Ejecución durante el despliegue: ``Principal``
    * Fuente del evento: ``De un formulario``
    * Tipo de evento: ``Al enviarse el formulario``
    * Notificar errores: ``Notificarme cada día``
